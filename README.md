## Избранные проекты

### 1. Яндекс.Самокат — Дипломный проект (Комплексное тестирование)

**Самый масштабный проект:** тестирование веб-приложения, мобильного приложения и API сервиса заказа самокатов.

- **Веб:** Тестирование форм оформления заказа (валидация полей Имя, Телефон, Адрес, Дата доставки, Комментарий). Найдены критические баги в валидации полей.
- **Мобильное приложение (Android):** Тестирование нотификаций, поведения при отсутствии интернета, переходов по уведомлениям. Найден критический баг с падением приложения при переходе по нотификации.
- **API:** Тестирование эндпоинтов создания курьера, получения заказа, удаления курьера. Найдены ошибки валидации на сервере (принятие некорректных данных).

**Стек:** Postman, Android Studio, Redmi Note 13 Pro (Android 15), Google Sheets, YouTrack

[`Дипломный проект →`](Проекты%20Яндекс%20Практикум/Тестирование/Диплом/README.md)

### 2. Яндекс Метро — Мобильное приложение (Android)

**Тестирование мобильного приложения** для навигации в метрополитене.

- Проведено **95 тестов**, из них **17 с ошибками** (средний приоритет).
- Применены виды тестирования: функциональное, тестирование прерываний, удобства использования, регрессионное.
- Составлен регрессионный чек-лист.
- Найдены баги, влияющие на пользовательский опыт, но не блокирующие основной функционал.

**Стек:** Android Studio, API 28 (Pie), Honor 8, Google Sheets

[`Яндекс Метро →`](Проекты%20Яндекс%20Практикум/Тестирование/Спринт%204/README.md)

### 3. Kittygram — Тестирование авторизации (Web)

**Тестирование системы авторизации** веб-приложения для обмена фотографиями котов.

- Разработана **декомпозиция логики** тестирования процесса авторизации.
- Проведен **анализ граничных значений** (Boundary Value Analysis) для полей Имя и Пароль.
- Составлены **тест-кейсы на верстку** (26 тестов) и **тест-кейсы на логику** (26 тестов).
- Найдены **критические баги:** авторизация с пробелами в логине/пароле, некорректный редирект авторизованного пользователя.

**Стек:** Google Sheets, Chrome DevTools, YouTrack

[`Kittygram →`](Проекты%20Яндекс%20Практикум/Тестирование/Спринт%202/README.md)

### 4. Проект "Место" — Регрессионное тестирование (Web)

**Регрессионное тестирование** социальной сети для публикации фотографий.

- Проведено тестирование функции редактирования профиля.
- Найдено **9 багов**, из них **3 блокирующих** (невозможность сохранить данные профиля).
- Составлены тест-кейсы и баг-репорты с подробным описанием шагов воспроизведения.
- Даны рекомендации: **не рекомендовано к релизу** из-за критических ошибок.

**Стек:** Yandex Browser, Windows 10, Google Sheets
